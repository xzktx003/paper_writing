# Reviewer Notes

This is a minimal, redistributable Paper Agent project for demonstration and review.

Suggested smoke test:

1. Copy `examples/demo-paper/` into the configured `papers/` directory.
2. Open the project from the Paper Agent project list.
3. Open `main.tex`.
4. Run citation cross-check against `references.bib`.
5. Compile the project with the available LaTeX engine.

No private data, API key, generated PDF, or local compile artifact is included.
