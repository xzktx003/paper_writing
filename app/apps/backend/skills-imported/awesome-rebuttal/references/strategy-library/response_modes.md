# Response Modes

| Mode | Use when | Risk | Required evidence |
|---|---|---|---|
| clarify_misunderstanding | Paper already answers concern | sounding dismissive | paper anchor |
| present_existing_evidence | Existing experiment/result supports claim | overclaiming | result/table/figure anchor |
| add_supplementary_experiment | Feasible experiment addresses decision-critical gap | negative/noisy result | experiment plan/result |
| acknowledge_limitation | Concern is valid and cannot be fully fixed | weakening paper | bounded limitation text |
| soften_claim | Reviewer flags overclaiming | losing contribution clarity | revised claim |
| revise_wording | Concern is clarity/presentation | superficiality | edit commitment |
| explain_tradeoff | Design choice has costs/benefits | seeming evasive | code/paper rationale |
| defer_intentionally | Not feasible/allowed in rebuttal | reviewer dissatisfaction | reason and future-work framing |
