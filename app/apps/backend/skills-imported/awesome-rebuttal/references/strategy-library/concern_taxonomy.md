# Concern Taxonomy

Use these categories as defaults; add project-specific categories when needed.

- novelty_or_incrementality
- missing_baseline
- dataset_bias_or_scope
- small_gain_or_practical_significance
- overclaiming
- statistical_significance
- reproducibility
- clarity_or_presentation
- method_validity
- theoretical_rigor
- efficiency_or_complexity
- ablation_or_sensitivity
- generalization
- venue_fit
- ethical_or_broader_impact
- minor_typo_or_format

Concern type:

- `misunderstanding`: reviewer missed or misread existing content.
- `evidence_gap`: claim may be true but evidence is insufficient.
- `true_limitation`: concern is valid; response should bound claims.
- `writing_issue`: fixable by clarification/paper edit.
- `rule_or_format_issue`: constrained by venue instructions.
