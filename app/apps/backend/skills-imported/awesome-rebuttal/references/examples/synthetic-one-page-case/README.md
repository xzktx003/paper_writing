# Synthetic One-Page Case

Use this case to test common-concern clustering and compression.

## Inputs

- Venue rules: user-confirmed one-page unified response.
- Reviews: four reviewers with overlapping novelty, ablation, and reproducibility comments.

## Expected outputs

- Common clusters, not reviewer-by-reviewer duplication.
- Space allocation table.
- AC-oriented summary decision if rules allow.
- Safety checklist showing all reviewer concerns covered or intentionally deferred.
