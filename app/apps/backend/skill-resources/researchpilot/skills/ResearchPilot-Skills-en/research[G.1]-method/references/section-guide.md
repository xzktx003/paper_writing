# Method 章节写作规范

`references/section-guide-source.md` 是来自 Master-cai method.md 的完整原文指南。
`references/examples/` 包含 Neural Body 三要素例子和模块设计模式。

核心规则：
1. 每个模块按"设计→动机→技术优势"三要素顺序写
2. 先写 Overview（含 pipeline 图引用），再写各模块，最后写训练目标
3. 每个公式后解释所有变量含义
4. 技术描述顺序：先"做什么"，再"为什么"，最后"公式"
5. 不写"Inspired by [x]"模糊表述，说清楚借鉴了哪个具体机制
