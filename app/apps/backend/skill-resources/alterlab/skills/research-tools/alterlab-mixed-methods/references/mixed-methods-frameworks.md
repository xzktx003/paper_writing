# Mixed Methods Frameworks — Reference Guide

## Major Design Typologies Compared

### Creswell & Plano Clark (2018) Typology

| Design | Notation | Timing | Priority | Integration Point |
|--------|----------|--------|----------|-------------------|
| Convergent | QUAL + QUAN | Concurrent | Equal | Interpretation |
| Explanatory Sequential | QUAN → qual | Sequential (quan first) | Quantitative | Connection |
| Exploratory Sequential | qual → QUAN | Sequential (qual first) | Quantitative | Connection |
| Embedded | QUAN(qual) or QUAL(quan) | Concurrent or sequential | One dominant | Embedding |
| Transformative | Framework[any design] | Varies | Varies | Framework-level |
| Multiphase | Multiple linked studies | Multi-project | Varies | Programmatic |

### Morse (2003) Typology

| Design | Notation | Description |
|--------|----------|-------------|
| Simultaneous QUAL-driven | QUAL + quan | Qualitative primary, quantitative supplementary, concurrent |
| Simultaneous QUAN-driven | QUAN + qual | Quantitative primary, qualitative supplementary, concurrent |
| Sequential QUAL-driven | QUAL → quan | Qualitative first, then quantitative follow-up |
| Sequential QUAN-driven | QUAN → qual | Quantitative first, then qualitative follow-up |

### Greene, Caracelli & Graham (1989) Purposes for Mixing

| Purpose | Definition | Example |
|---------|-----------|---------|
| Triangulation | Seeking convergence of results from different methods studying the same phenomenon | Comparing survey results with interview findings on job satisfaction |
| Complementarity | Using results from one method to elaborate, enhance, or clarify results from another | Using interviews to add depth to experimental findings |
| Development | Using results from one method to develop or inform the other | Using focus groups to develop survey items |
| Initiation | Discovering paradoxes, contradictions, and new perspectives | Finding that satisfaction scores are high but narratives describe dissatisfaction |
| Expansion | Extending the breadth and range of inquiry by using different methods for different components | Survey for prevalence, interviews for process, observation for behavior |

## Integration Techniques — Detailed Reference

### Merging Strategies

#### Data Transformation Matrix

| From → To | Technique | When to Use | Caution |
|-----------|-----------|-------------|---------|
| Qual → Quan (Quantitizing) | Theme frequency counts | When you need to statistically compare theme prevalence across groups | Reduces richness; report alongside original qualitative data |
| Qual → Quan (Quantitizing) | Binary coding (present/absent) | When testing whether specific themes associate with quantitative outcomes | Loss of nuance in theme intensity and meaning |
| Qual → Quan (Quantitizing) | Rating scales from qual data | When experts rate qualitative data on a quantitative dimension | Requires inter-rater reliability assessment |
| Quan → Qual (Qualitizing) | Narrative profiles from clusters | When you want to describe the "story" of each statistical cluster | Profile may oversimplify within-cluster variation |
| Quan → Qual (Qualitizing) | Case descriptions from factor scores | When you want to illustrate what factor combinations look like in practice | Requires carefully selected exemplar cases |

#### Joint Display Construction Guide

**Step 1: Identify the organizing framework**
- By research question (rows = questions, columns = strand findings)
- By variable/theme (rows = key constructs, columns = strand findings)
- By case (rows = individual participants, columns = data types)
- By time/phase (rows = time points, columns = data types)

**Step 2: Populate both columns independently**
- Enter quantitative findings (statistics, effect sizes, significance)
- Enter qualitative findings (themes, categories, exemplar quotes)

**Step 3: Add the integration column**
- Assess convergence, complementarity, or divergence
- Write meta-inferences for each row
- Note where additional data is needed

**Step 4: Develop overarching meta-inferences**
- Synthesize across rows
- Identify patterns in convergence/divergence
- State the "value added" by integration

### Connecting Strategies for Sequential Designs

#### Explanatory Sequential: QUAN → qual Connection Points

| Quantitative Result | Qualitative Follow-Up Strategy |
|--------------------|---------------------------------|
| Significant predictor variable | Interview participants about their experience with that variable |
| Unexpected non-significant finding | Explore why expected relationship was not found |
| Significant interaction effect | Interview from each subgroup to understand different mechanisms |
| Outlier cases | Interview outliers to understand why they deviate |
| Significant group differences | Interview from each group to explain the difference |
| Low model fit (R² < .20) | Explore unmeasured variables through open-ended inquiry |

#### Exploratory Sequential: qual → QUAN Connection Points

| Qualitative Finding | Quantitative Development Strategy |
|--------------------|------------------------------------|
| Emergent theme with clear dimensions | Develop multi-item scale for the construct |
| Participant-generated categories | Create categorical survey items using participant language |
| Process model with sequential stages | Develop measures for each stage, test sequential relationships |
| Contextual moderators mentioned | Include as moderator variables in quantitative model |
| Unexpected construct | Search for existing validated measures; adapt or develop new |

## Quality Assessment Frameworks

### GRAMMS (Good Reporting of A Mixed Methods Study)

Checklist for manuscripts:
1. Describe the justification for using a mixed methods approach
2. Describe the design type (with citation to methodology literature)
3. Describe each method in terms of sampling, data collection, and analysis
4. Describe where integration has occurred, how it has occurred, and who has participated in it
5. Describe any limitation of one method associated with the presence of the other method
6. Describe any insights gained from mixing or integrating methods

### MMAT (Mixed Methods Appraisal Tool) — Version 2018

Screening questions (for all studies):
- S1. Are there clear research questions?
- S2. Do the collected data allow to address the research questions?

Mixed methods criteria:
- 5.1. Is there an adequate rationale for using a mixed methods design?
- 5.2. Are the different components of the study effectively integrated?
- 5.3. Are the outputs of the integration adequately interpreted?
- 5.4. Are divergences and inconsistencies between quantitative and qualitative results adequately addressed?
- 5.5. Do the different components of the study adhere to the quality criteria of each tradition?

## Philosophical Worldviews for Mixed Methods

| Worldview | Core Tenets | Implications for Mixed Methods | Key Authors |
|-----------|-------------|-------------------------------|-------------|
| Pragmatism | Knowledge is constructed through action; focus on consequences and "what works" | Methods selected for fitness to research question; paradigm debates secondary | Dewey, James, Rorty, Morgan |
| Critical Realism | Reality exists independently but our knowledge of it is fallible and theory-laden | Methods triangulate around a stratified ontology; retroduction bridges methods | Bhaskar, Maxwell, Zachariadis |
| Dialectical Pluralism | Multiple paradigms can be held in productive tension | Seek dialectical insights from paradigm contrasts | Johnson, Onwuegbuzie |
| Transformative | Research serves social justice; centers marginalized perspectives | Design decisions filtered through equity framework | Mertens, Sweetman |
| Pragmatism of the Middle | Balance between pure pragmatism and philosophical rigor | Practical decisions within philosophical awareness | Teddlie & Tashakkori |
